Distributed Supermarket System

This is a multi-threads client system and uploaded 300,000 purchase data with a throughput of 15000/s.

It was built on AWS EC2 using Tomcat server, and connected to Amazon RDS. I optimized the system using Load balancer and database sharding. I also used RammitMQ and the message-broker to increase the speed.
